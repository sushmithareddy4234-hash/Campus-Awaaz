#  Campus Awaaz

Campus Awaaz is a platform where students can report issues happening in their college and support (upvote) them. It ensures anonymity while allowing students to raise concerns safely.

---

##  Features

-  Report campus issues  
-  Vote for issues (with reason/proof)  
-  Anonymous participation (no name required)  
-  Upload proof (images/videos)  
-  Admin can resolve issues with explanation  
-  Separate "Resolved Issues" section  

---

##  Problem It Solves

Students often hesitate to report issues like ragging, infrastructure problems, or faculty concerns. This platform gives them a **safe and anonymous voice**.

---

##  Tech Stack

- Frontend: React.js + Tailwind CSS  
- Backend: Node.js + Express.js  
- Database: (Can be added later like MongoDB)  

---

##  How to Run Locally

### 1. Clone the repo
git clone https://github.com/sushmithareddy4234-hash/Campus-Awaaz.git

### 2. Backend setup
cd backend  
npm install  
node server.js  

### 3. Frontend setup
cd frontend  
npm install  
npm start
